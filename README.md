## How this gets deployed

This repo is the **deploy target**, not the source of truth. The site is
built from `Forgesaroj/Standlone-Lumora-ERP/client/src/landing` as an
isolated marketing-only Vite bundle and pushed here on demand.

policy enforce this. See [CONTRIBUTING.md](./CONTRIBUTING.md) for the
full policy.
