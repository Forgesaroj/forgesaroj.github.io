A global, multi-country ERP — built to run on the cloud or on a Raspberry Pi.

Standalone ERP is a modular accounting and operations platform with 99 business modules, a strict double-entry accounting engine, and a plugin-based country-pack architecture. One codebase. Two deployment targets: hosted SaaS, or a self-contained box on the customer's premises.

See what it does Tech stack

What's inside

Built around a strict accounting engine, with everything else layered on top.

Strict double-entry engine

Every posting balances per company, per currency. Cost-centre allocations sum to 100% or the posting refuses. Reversal preserves audit history.

Multi-tenant, multi-company

Row-level security at the database. Companies live under tenants; cross-company access is explicit and audited.

Country packs as plugins

Tax rules, statutory reports, and chart templates ship per country (India, Nepal, …) and can be installed or removed without touching core.

Dual deployment

The same binary runs as a hosted SaaS service or as a single-file deployment on a Raspberry Pi for on-premise customers.

Tally-prime familiar UX

Keyboard-first workflows, F-key navigation, and voucher entry that anyone coming from Tally can use on day one.

Audit-grade integrity

Hash-chained activity log, period locks, balance assertions, idempotent imports. The data is supposed to be defensible.
